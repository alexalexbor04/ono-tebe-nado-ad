# Оно тебе надо — аукцион вещей

Учебный проект по вёрстке: семантическая HTML‑структура и стилизация по макету.

## Ссылка на проект

https://github.com/alexalexbor04/ono-tebe-nado-ad

## Как запустить

- Откройте файл `index.html` в браузере.

## Структура проекта

- `index.html` — разметка страницы
- `styles/global.css` — базовые (глобальные) стили и сброс
- `styles/style.css` — основные стили проекта
- `fonts/` — шрифты и подключение (`fonts.css`)
- `images/` — изображения и иконки

## Использованные технологии

- HTML
- CSS
